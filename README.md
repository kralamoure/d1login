# d1login

`d1login` is a login server for Dofus 1.

## Requirements

- [Git](https://git-scm.com/)
- [Go](https://golang.org/)

## Build

```sh
git clone https://github.com/kralamoure/d1login
cd d1login
go build
```

## Installation

```sh
go get github.com/kralamoure/d1login/cmd/d1login
```

## Usage

```sh
d1login --help
```
